# Decidim::BudgetsEnhanced

Budgets nehanced.

## Usage

BudgetsEnhanced will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'decidim-budgets_enhanced
```

And then execute:

```bash
bundle
```
